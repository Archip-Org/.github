## 📖 Sobre o Projeto
A **Archip** é uma iniciativa brasileira de código aberto que visa projetar e desenvolver processadores baseados na arquitetura **RISC-V** para notebooks.

> ⚠️ **Status: Em desenvolvimento ativo (Alpha)**. 
> Estamos na fase inicial de aprendizado e implementação com a placa **Tang Nano 20k** (FPGA). Este não é um produto finalizado, mas uma jornada aberta para democratizar o design de chips.

## 🎯 Nossa Missão
- Criar um núcleo RISC-V completamente funcional e otimizado.
- Documentar cada etapa do desenvolvimento em português e inglês.
- Construir uma comunidade global de engenheiros e entusiastas de hardware.

## 🗺️ Roadmap (O que estamos fazendo)
- [ X ] Configuração do toolchain (Yosys, Nextpnr).
- [ X ] Implementação da ULA (Unidade Lógica e Aritmética).
- [ ] Implementação do Banco de Registradores.
- [ ] Máquina de Estados e Pipeline inicial.
- [ ] Testes em hardware real (Tang Nano 20k).
